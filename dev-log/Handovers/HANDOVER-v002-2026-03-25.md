## Decisions Made
- **Electric teal locked** — `#05D4B8` for `--teal-hero`. May need to dial back brightness — easy one-line change in globals.css
- **Token set finalised** — teal × 5, purple × 5, UI surfaces × 6. No KFG tokens (cards borrow from strength family hero). No mid-purple token (it's just purple-growing).
- **Hero and nailing-it are separate tokens** — same hex value, different semantic roles. Hero = brand usage, Nailing it = score state. Independent so they can diverge later.
- **Dark mode removed from globals.css for now** — to be reintroduced when design is stable. `.dark {}` block deleted.
- **COLORS key naming** — sub-keys named after score states (notYet / learning / growing / hero+nailing) so intent is always clear at point of use
- **Dashboard first, home page second** — home page borrows dashboard layout, so dashboard must be built first. GrrowCircle upgrades unlock both.
