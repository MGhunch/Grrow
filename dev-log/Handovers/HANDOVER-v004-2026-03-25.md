# HANDOVER v001 — 2026-03-25

## Project
Grrow — a structured professional development tool mapping skills across four core strengths and four career stages. The circle is the product.

## Session Focus
Built a 3D-enhanced version of the Grrow circle component with gradients, shadows, and beveled edges — inspired by a tactile wooden/plastic puzzle reference image. Created as a new component to avoid breaking the existing one.

## Code Updated
- `GrrowCircle3D.tsx` (new file — drop into `src/components/`)

## Decisions Made
- **New component, not a modification** — keeps the original `GrrowCircle.tsx` intact for safe A/B comparison
- **All colours from constants.ts** — no hardcoded hex in the component body; pulls from `COLORS` object
- **Radial gradients for depth** — 8 total (4 score states × 2 families), lighter at outer edge, richer at inner for "raised segment" look
- **1° gaps between segments** — arcs span 88° instead of 90°, lets shadow breathe between pieces
- **Drop shadow on whole circle** — single `<feDropShadow>` filter, stronger in dark mode (0.4 opacity) vs light (0.2)
- **Highlight rings** — thin white strokes at 12% opacity on ring boundaries for subtle bevel
- **Thinner separators** — 1.5px instead of 2.5px since gaps now do some of that visual work
- **Same props API** — identical to original component, so it's a drop-in replacement
