## Decisions Made

- **Teal hero colour changed** — `#18B99A` replaced with `#05D4B8` (Electric). Current teal judged "muddy" against deep purple. New value holds intensity through the score ramp.
- **Purple unchanged** — `#4C3FA0` locked. No case made to change it.
- **Full teal ramp for Electric:**
  - Not yet: `#D6FBF7`
  - Learning: `#9AF0E8`
  - Growing: `#3DE8D4`
  - Nailing it: `#05D4B8`
- **3D circle treatment rejected** — not buildable as interactive SVG. Segment gaps from the 3D renders worth stealing for the flat SVG.
- **Dark mode disabled for now** — to be reintroduced later when design is stable.
- **KFG cards added to dashboard** — Keep / Focus / Grow cards below the circle are the primary content below the fold. Not optional.
- **KFG card tag = ring name** — shows "Essentials / Exploring / Influencing / Leading", not strength name.
- **Button copy** — "How are you growing?" 
- **Accent lines on KFG cards** — curved corner accent in category colour. Keep them.
- **Home page (logged out)** — same layout as dashboard. Three cards show Meaningful / Manageable / Measurable instead of KFG. Demo circle animation in place of live scores.
- **GrrowCircle modes** — `mode="demo"` for landing page (scripted animation), `mode="live"` for dashboard (real scores + KFG glow).
- **CSS token architecture** — globals.css should be single source of truth for all colour tokens. tailwind.config.ts and constants.ts should consume variables, not hardcode hex. Not yet implemented — Session 1 fixes this.
